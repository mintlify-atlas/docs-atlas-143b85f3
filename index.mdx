      Quick Start

    <p className="text-lg text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running in minutes with our starter template
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Clone the starter template to your local machine:
      
      ```bash
      git clone https://github.com/nextjs/saas-starter
      cd saas-starter
      pnpm install
      ```
    </Step>
    
    <Step title="Set up your database">
      Create your environment file and configure your database:
      
      ```bash
      pnpm db:setup
      pnpm db:migrate
      pnpm db:seed
      ```
      
      This creates a test user with email `test@test.com` and password `admin123`.
    </Step>
    
    <Step title="Configure Stripe">
      Set up your Stripe keys in the `.env` file:
      
      ```bash
      STRIPE_SECRET_KEY=sk_test_***
      STRIPE_WEBHOOK_SECRET=whsec_***
      ```
      
      <Tip>Use the Stripe CLI to listen for webhooks locally: `stripe listen --forward-to localhost:3000/api/stripe/webhook`</Tip>
    </Step>
    
    <Step title="Start developing">
      Run the development server:
      
      ```bash
      pnpm dev
      ```
      
      Open [http://localhost:3000](http://localhost:3000) to see your app running.
    </Step>
  </Steps>
</div>
